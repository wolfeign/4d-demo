# 4D Demo

4次元の超立方体を表示するデモ。

A demo showing a 4-dimensional hypercube.

## デモ (Demo)

https://wolfeign.github.io/4d-demo/

## 使用方法 (Usage)

マウスの左ボタンをドラッグすると図形を回転できる。

You can rotate the shape by dragging the left mouse button.
